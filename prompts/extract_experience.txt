# Prompt: extract_experience
# Purpose: Extract the minimum years of experience required from a job description.
# Expected response: A single integer, or "none" if not mentioned.

Read the following job description and determine the minimum years of professional experience required.

Job Description:
{{JOB_DESCRIPTION}}

Rules:
- Return ONLY a single integer (e.g. 3, 5, 10)
- If the description says "3+ years", return 3
- If the description says "3-5 years", return 3 (the minimum)
- If no years of experience are mentioned, return the word "none"
- Do not include any other text in your response
