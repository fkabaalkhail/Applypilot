# Prompt: match_job
# Purpose: Analyze how well a resume matches a job posting.
# Expected JSON response shape:
# {
#   "match_score": 85,
#   "requirements": [
#     {"requirement": "3+ years Python experience", "met": true},
#     {"requirement": "AWS certification", "met": false}
#   ],
#   "summary": "Brief 2-sentence summary of the role",
#   "salary_range": "$100k - $150k" or "" if not mentioned,
#   "company_size": "101-500 employees" or "" if not mentioned,
#   "company_description": "One sentence about what the company does"
# }

You are a job matching assistant. Analyze how well the candidate's resume matches this job posting.

RESUME:
{{RESUME_TEXT}}

JOB TITLE: {{JOB_TITLE}}
COMPANY: {{JOB_COMPANY}}
JOB DESCRIPTION:
{{JOB_DESCRIPTION}}

Return ONLY valid JSON with these keys:
- match_score: integer 0-100 representing overall fit percentage
- requirements: array of objects with "requirement" (string) and "met" (boolean). Extract each requirement from the job description and check if the resume satisfies it.
- summary: 2-sentence summary of what the role involves
- salary_range: salary range if mentioned in the description, empty string if not
- company_size: company size if mentioned, empty string if not
- company_description: one sentence about what the company does, based on the job description
